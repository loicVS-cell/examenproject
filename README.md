# examenproject
Een Java-applicatie voor het registreren van passagiers en het verkopen van treintickets voor Euromoon.
Een Java-applicatie voor het beheren van treinen, reizen, personeel, passagiers en tickets.

## Functionaliteiten

- Passagiers registreren
- Reizen aanmaken en beheren
- Treinen koppelen aan reizen
- Tickets verkopen
- Boardinglijsten genereren

## Structuur

- `src/main/java/euromoon/model/`: Classes voor data (bv. `Persoon`, `Reis`, `Trein`).
- `src/main/java/euromoon/service/`: Logica voor reisbeheer, ticketverkoop, etc.
- `src/main/java/euromoon/data/`: Voorbeelddata voor de applicatie.
- `src/main/java/euromoon/`: Hoofdprogramma met menu.

## Gebruikte Bronnen

- Java Documentatie: [Oracle Docs](https://docs.oracle.com/en/java/)
- Mistral AI: Voor het genereren van voorbeelddata.
